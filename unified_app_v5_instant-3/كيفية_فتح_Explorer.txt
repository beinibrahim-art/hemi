═══════════════════════════════════════════════════════════
   📖 كيفية فتح Explorer في Cursor
═══════════════════════════════════════════════════════════

🔍 الطريقة 1: من الشريط الجانبي (الأسهل)
───────────────────────────────────────────────────────────
1. انظر إلى الجانب الأيسر من نافذة Cursor
2. ستجد أيقونات رأسية في الشريط الجانبي
3. ابحث عن أيقونة المجلدات 📁 (أو File Explorer)
4. انقر عليها

الأيقونة عادة تكون:
   📁 File Explorer
   أو
   📂 Explorer
   أو
   📁 Files

───────────────────────────────────────────────────────────

🔍 الطريقة 2: من القائمة العلوية
───────────────────────────────────────────────────────────
1. انظر إلى القائمة العلوية في Cursor
2. ابحث عن: View
3. انقر على View
4. ابحث عن: Explorer أو File Explorer
5. انقر عليه

أو:
   View → Explorer
   أو
   View → File Explorer

───────────────────────────────────────────────────────────

🔍 الطريقة 3: من لوحة المفاتيح (الأسرع)
───────────────────────────────────────────────────────────
اضغط:
   Cmd+Shift+E  (على Mac)
   أو
   Ctrl+Shift+E  (على Windows/Linux)

───────────────────────────────────────────────────────────

🔍 الطريقة 4: من Command Palette
───────────────────────────────────────────────────────────
1. اضغط: Cmd+Shift+P (Mac) أو Ctrl+Shift+P (Windows)
2. اكتب: "View: Show Explorer"
3. اضغط Enter

───────────────────────────────────────────────────────────

📍 بعد فتح Explorer:
───────────────────────────────────────────────────────────
سترى شجرة المجلدات على اليسار:
   📁 unified_app_v5_instant-3
      📁 templates
         📄 index.html  ← هذا هو الملف المعدل

انقر على unified_app_v5_instant-3
ثم انقر على templates
ثم انقر على index.html

═══════════════════════════════════════════════════════════

💡 نصائح:
───────────────────────────────────────────────────────────
• إذا لم ترَ الشريط الجانبي، اضغط Cmd+B (Mac) لإظهاره
• يمكنك أيضاً استخدام البحث: Cmd+P ثم اكتب "index.html"
• Explorer يظهر عادة كأول أيقونة في الشريط الجانبي
